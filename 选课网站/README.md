# 选课网站

这是一个功能完整的在线选课系统，使用纯HTML、CSS和JavaScript开发，无需任何框架。

## 功能特点

- 📚 **课程浏览** - 查看所有可选课程的基本信息
- 🔍 **搜索筛选** - 按课程名称、教师、分类、时间和学分筛选课程
- 📖 **课程详情** - 查看课程的详细介绍和评价
- ✅ **选课功能** - 添加和取消选择课程
- 📊 **我的选课** - 查看已选课程和总学分统计
- 💾 **数据持久化** - 选课状态保存在本地存储中
- 📱 **响应式设计** - 适配不同屏幕尺寸的设备

## 技术栈

- HTML5
- CSS3 (使用CSS变量和Grid布局)
- 纯JavaScript (ES6+)
- Font Awesome 图标库

## 项目结构

```
选课网站/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # 交互逻辑
└── README.md           # 项目说明
```

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 浏览课程列表，使用搜索和筛选功能找到感兴趣的课程
3. 点击"详情"查看课程详细介绍
4. 点击"选课"将课程添加到"我的选课"
5. 在"我的选课"页面查看已选课程和总学分
6. 可以随时取消已选课程

## 部署说明

### 使用Netlify部署

1. 将代码上传到GitHub仓库
2. 在Netlify中连接GitHub仓库
3. 设置构建命令为空（因为是静态网站）
4. 设置发布目录为根目录
5. 点击部署即可

### 其他部署方式

任何支持静态网站的服务都可以部署此项目，例如：
- Vercel
- GitHub Pages
- 阿里云OSS
- 腾讯云COS
